# Rock-Paper-Scissor

## Description
A simple implementation of the classic Rock-Paper-Scissor game.

## Features
- Play Rock-Paper-Scissor against the computer.
- Keep track of scores.
- Simple and intuitive user interface.


## Usage
Run the game:
```sh
python game.py
```
